I didn't find much information in any forum (the ones I know) about using Vue xStream in C4D apart from questions if it works and problems.


But there seems to be a lot of interest, so I'm writing this.


As I had to renew my educational license on Vue , for an extra $50 I jumped to Vue xStream


First the setting 

Vue 8 xStream and C4D R11.5

Mac OS X 11.6.2 with 3GB of memory on an old iMac (2.16GHz Intel Core 2 Duo)


My first impression : Works well


Caveat : Vue 8 and plug-in for C4D are only in 32 bits.





Once you get pass these menus you get exactly the same editor windows as if you were in Vue (terrain editor, etc).


Performance : I used the standard Vue procedural terrain Blobby.   Render at 960x540


1. Render in Vue 8 xStream. Render quality : Final   Time : 3 min 6 sec



















 

Vue xStream with Cinema 4D

2. Render in C4D  Anti-Aliasing : geometry  1 m 32 sec

3. Render in C4D  Anti-Aliasing : Best    31 min 18 sec

So, the render time in Vue xStream alone is really good for the quality. Inside C4D at geometry quality the edges are really jaggy . At Best quality the result is as good as in Vue xStream alone but the render time is 10 folds.
